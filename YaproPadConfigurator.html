Sidebar And Configuration Settings

This is the Yap-ro Pad Configuration webpage where you can set the modes and functions of the various inputs such as sliders, and keyboard buttons. Starting with the sidebar there is the option to upload an existing configuration file for modification and an option to download a config File with the options as they have been configured on the webpage. The settings button will alow you to configure how many sliders and input Keyboards your specific Yap-ro Pad has connected and to configure the colors supported by your slider displays. The sliders within the configurator should match the number of sliders that you have on your device and will display the Name of what the slider controlls such as "Discord" the icon of the slider the current volume and whether or not the current slider is muted. You can change the Volume level and mute status by either manually ajusting the slider or mute button on the device or clicking its representation within the configurator. The configurator also allows you to change the name and icon of a slider simply by clicking on either the name or the icon within the configurator.

Keyboards

The keyboard configuration section of the configurator allows you to select which keyboard you are configuring and then set the function of each key. the sidebar displays a list of configured keyboards and hilights the currently selected keyboard.
